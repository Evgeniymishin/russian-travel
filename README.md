# Проект: Путешествие по России

### Описание
Учебный проект от "Яндекс.Практикум".
Адаптивный одностраничный сайт о путешествия по России.

### Используемые технологии

* Grid, Flex верстка
* Медизапросы
* Методология БЭМ
